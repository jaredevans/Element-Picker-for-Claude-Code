Bookmarklet Tool

Element Picker
for Claude Code

Click elements on any page, annotate each one, and collect them all into a single clipboard-ready list to paste into Claude Code.

Install

⊕ Element Picker Drag this button to your bookmarks bar. Or right-click → Bookmark this link.
1

Activate the picker

Navigate to any page, then click the bookmarklet. A red banner and an Annotations panel appear.

2

Click & annotate elements

Click an element → it highlights red → type your comment → press Enter or click Add to List. The annotation appears in the panel. Repeat for as many elements as you want.

3

Review the list

The Annotations panel shows all your picks. Remove any with the × button. Minimize the panel with — if it’s in your way.

4

Copy & paste into Claude Code

Click Done & Copy (or press Esc) to copy all annotations to clipboard and exit. Click Copy All to copy without exiting.
